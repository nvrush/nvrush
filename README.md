```sh
mkdir -p ~/.config/Codeotus && cd ~/.config/Codeotus

NVIM_APPNAME=Codeotus nvim
```

**Alias it:**
```sh
echo "alias cod='NVIM_APPNAME=Codeotus nvim'" >> ~/.bashrc

```

Launch: `cod`

**Uninstall:**
```sh
rm -rf ~/.config/Codeotus/ ~/.local/share/Codeotus/
```
